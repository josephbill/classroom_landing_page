## GIT / GITHUB 
Git  : software system to support version control 
Github : platform to connect to our repositories : storage location for 
your commits 

## 1. When work new : initial commit 
1. git init : initializes an empty .git folder : local repo
2. git remote add origin remoteURL : (ssh) (https) : simply connects loc
al repo to remote repository 
3. git status : check which files are ready to be tracked 
red: not ready green: ready 
4. git add nameoffile 
   git add .  - add everything 
5. git commit -m "message tag" : created a snapshot of your project 
6. git push -u origin branchName (master or main)
   - ssh - connection is auto 
   - https - password token (personal access token)
